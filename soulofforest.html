Soul of the Forest

WolverineSoft Studio, Quality Assurance Programmer, Winter 2024

Project Description

In this single player, action-adventure game, the player takes the role of the Druid, an estranged creature who must save those that isolate him from an unforeseen threat.

Game and Contribution Details

  • Top-down fantasy RPG inspired by Undertale, Franken, and others
  • Worked in the programming department and quality assurance sub department to develop the studio's first big RPG in Unity
  • Primarily implemented game UI designs, integrating them with the game's save and quest systems
  • Collaborated regularly with the narrative, art, and audio departments to create and debug in game effects


Technologies Used: Unity, C#, Jira, Bitbucket, Confluence
